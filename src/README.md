# Kalaikatha

Indian heritage crafts platform. Artisan-to-customer ecosystem with AI assistance.

---

## Quick Start

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173` with mock data.

## Key Features

✅ Multi-language (8 Indian languages, download on selection)  
✅ Low-end optimized (2G, 1GB RAM, 92% image compression)  
✅ Voice-first (input everywhere, TTS for illiterate users)  
✅ Azure AI (pricing, negotiation, marketing, trade secrets)  
✅ Offline-first (works without internet)

---

## Tech Stack

React + Vite + TypeScript + Tailwind v4  
Azure Functions + SQL + Blob Storage + OpenAI
